Current Layout Details:

  • Layer 0 (Base): Neo-inspired layout with German umlauts (Ü, Ö, Ä) and ß on dedicated keys
  • Home Row Modifiers: GUI/Alt/Shift/Ctrl on U,I,A,E (left) and S,N,R,T,D (right)
  • Layer 1: Navigation keys on left, numpad on right (0 on right thumb)
  • Layer 2: Symbols and special characters
  • German Support: All keys use DE_ prefix for proper QWERTZ OS mapping
  • Build status should now be successful with the fixed &kp DE_SZ syntax
